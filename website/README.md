# MIGHT Website (Next.js)

Website promosi bertema pixelated untuk aplikasi MIGHT, berisi fitur, kegunaan, dan tautan unduhan.

## Menjalankan Lokal

1. Pastikan Node.js 18+ terpasang.
2. Masuk ke folder `website/`.
3. Jalankan:

```bash
npm install
npm run dev
```

Akses di `http://localhost:5173`.

## Deploy ke Vercel

- Buat project baru di Vercel.
- Pilih repository ini, set **Root Directory** ke `website/`.
- Framework akan terdeteksi otomatis (Next.js). Build & deploy akan berjalan.

## Kustomisasi

- Ubah teks dan konten di `app/page.tsx`.
- Ubah warna/tema di `app/globals.css`.
- Tambahkan aset (logo, ilustrasi) ke `public/` jika diperlukan.
